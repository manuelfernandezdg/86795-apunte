AWC | Clase 6: Introducción a Javascript

Javascript

JavaScript es el lenguaje de programación que aporta dinamismo e interactividad a las páginas web, permitiendo ejecutar código directamente en el navegador del usuario. Actúa como el motor que manipula el DOM generado a partir del HTML, modificando nodos, agregando o eliminando elementos, cambiando estilos definidos por CSS, respondiendo a eventos del usuario (como clics, movimientos del ratón o entradas en formularios) y realizando operaciones lógicas o cálculos en tiempo real.

Gracias a JavaScript, una página inicialmente estática se transforma en una aplicación rica y responsiva, capaz de actualizar contenido sin recargar la página completa, comunicarse con servidores mediante peticiones AJAX o Fetch, y ejecutar lógica compleja, convirtiéndose en el pilar fundamental del desarrollo front-end moderno junto con HTML y CSS.

Manual de JS en la W3C

Objetos

Un objeto en Javascript es la representación de una unidad básica de organización. Un objeto es una colección de datos y funcionalidades relacionadas entre sí.

Dentro de un objeto encontramos:

  • Datos: llamados propiedades
  • Acciones: llamadas métodos

Si nuestro objeto fuese una persona, sus propiedades serían, por ejemplo, persona.edad, persona.altura, persona.cuadroDeFutbol.

Siguiendo el ejemplo, si tuviésemos métodos asociados a dicha persona, los mismos refieren a acciones, por ejemplo persona.caminar(), persona.dormir(), persona.comer().

Características de los objetos en Javascript

  • Un objeto puede tener cualquier cantidad de propiedades y métodos.
  • Las propiedades puede contenedor cualquier tipo de dato: strings, números, booleanos, arrays y también, otros objetos.
  • Los métodos son funciones guardadas dentro del objeto.
  • Los objetos son dinámicos: podemos agregar, modificar o eliminar propiedades y métodos en cualquier momento.

¿Todo en Javascript es un objeto?

Si. Por ejemplo:

  • Un elemento dentro de mi HTML, por ejemplo, un div es un objeto.
  • Un array [].
  • Una función.
  • document, window, console, son todos objetos.

Propiedades

En Javascript las propiedades son valores asociados a un objeto. Por ejemplo, si quieremos obtener el nombre de un usuario vamos a escribir usuario.nombre, donde usuario es el objeto y nombre su propiedad.

Las propiedades sirven para guardar o representar datos.

Las propiedades:

  • Se leen y escriben directamente.
  • Algunas son de sólo lectura, como por ejemplo:
    • element.clientWidth
    • elemento.offsetHeight
    • array.length
  • Pueden contener cualquier tipo de dato: string, número, objeto, booleano, arrays.

Métodos

Los métodos son funciones que pertenecen a un objeto. Al ser funciones siempre se ejecutan con paréntesis, por ejemplo getElementById(). Pueden recibir argumentos (parámetros). Generalmente producen un efecto, cambiar algo en el DOM, agregar un evento, modificar un array, entre otros.

Variables y Constantes en JS

Las variables son contenedores que nos permiten guardar valores. En JS tenemos una palabra reservada let para declarar nuestras variables. La palabra reservada const sirve para declarar constantes.

La diferencia entre las variables y constantes es que a las variables podemos reasignarles su valor.

Para asignar un nombre a las variables podemos utilizar:

  • Caracteres de la a-z / A-Z
  • Números 0-9
  • _ guión bajo

Uso del "Camel Case". Cuando el nombre de la variable contiene más de una palabra o término, es recomendable escribir la primera letra de cada término en mayúscula. Por ejemplo: precioArticuloPrincipal.

¿Cuándo usar const y cuándo usar let?

La regla principal que utilizamos es: utilizamos const de manera estándar y solamente utilizamos let cuando el valor de la variable va a cambiar a futuro.

  • En caso de duda: const
  • Utilizamos let en caso de que el valor de la variable será reasignado.
  • Casi jamás utilizamos var porque es antiguo y cayó en deshuso.
  • Declaramos las variables lo más cerca posible de dónde las utilizaremos (scope o alcance reducido).
